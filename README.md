# Word Frequency Analyzer

A Python based terminal application that takes raw text, cleans it of punctuation, and generates data reports on word usage. Users can input text directly or load a file, filter out common stopwords, and export a cleanly formatted frequency table. All logic is neatly organized into functions and controlled via an interactive main menu loop.

### Features
* **Dual Input Methods:** Read text from direct terminal input or load from a local `.txt` file.
* **Automated Text Cleaning:** Automatically normalizes text by removing punctuation and converting to lowercase.
* **Stopword Filtering:** Option to filter out common English stopwords for cleaner data.
* **Top 10 Report:** Displays a formatted table of the most frequently used words alongside overall word counts.
* **Targeted Word Search:** Search the exact frequency of a specific word within the text.
* **Data Export:** Export the full, sorted word frequency table to a physical `report.txt` file.

### How to Run Locally

1. **Prerequisites:** Ensure you have Python 3 installed on your computer.
2. **Clone the Repository:**  clone the repository on your terminal using `git clone`
3. **Navigate to the Directory:** use `cd word-frequency-analyzer` to navigate into the directory.
4. **Run the Application:** use python analyzer.py to run python file in terminal.
5. **Follow the Prompts:** Use the interactive terminal  menu to input your text and generate reports.
